# AI-Powered-Student-Learning-Recommendation-System
Proyek "AI-Powered Student Learning Recommendation System" dikembangkan untuk mengatasi permasalahan mahasiswa dalam memilih materi pembelajaran yang sesuai dengan performa akademis mereka. Saat ini, data Learning Management System (LMS) belum dimanfaatkan secara optimal untuk memberikan bimbingan personal.

How to use: 
run : python api_server.py

jika error maka run ini run pip install flask flask-cors pandas scikit-learn
